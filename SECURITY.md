# Security Policy

## Supported Versions

This project follows semantic versioning. Below are the currently supported versions receiving security updates:

| Version   | Supported          |
| --------- | ------------------ |
| 12.x      | :white_check_mark: |
| 6.x       | :white_check_mark: |
| 5.1.x     | :white_check_mark: |
| ≤ 5.0.x   | :x:                |

> Support is provided for the latest Laravel 12.x stack with Node.js 22, PHP 8.2, and React 19.

## Reporting a Vulnerability

If you discover a security vulnerability, please follow the steps below:

1. **Report it privately** by emailing: `rictmsfo4a@dswd.gov.ph`  
   *(Replace this with your actual contact email)*

2. Include the following details:
   - Steps to reproduce the issue
   - Affected version(s) or package(s)
   - Description of the potential impact

3. Expect a response within **2–3 business days**.

4. Once confirmed:
   - Valid issues will receive a patch or workaround.
   - You’ll be notified about the resolution status.

We appreciate responsible disclosure and aim to keep all supported versions safe and secure.
